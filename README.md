## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ yarn start

# debug mode
$ yarn start:debug

# production mode
$ yarn start:prod
```

## Testing

<p>Mail templates may be previewed with live changes in a browser. A local mail server is also available for sending and receiving of emails.</p>

```bash
# Mail template live preview
$ yarn mail:dev

# Local mail server
$ yarn mail:up
```
